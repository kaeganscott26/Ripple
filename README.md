# RIPPLE

_Status: Game Repo | Living Board Prototype | Source-Fueled Simulation_

Ripple is a standalone 2D board game and AI civilization simulation built from symbolic source material.

It is not INTERVENTION.

INTERVENTION remains the ARG / literary source archive. Ripple uses the copied source material as fuel: chapters become symbolic room functions, artifacts become board objects, and characters become memory-driven agents.

The goal is not to replay the story.

The goal is to let a new reality generate from it.

---

## What Ripple Is

Ripple is a turn-based living board game where observation changes the world.

The player enters as the Observer. The Observer does not command every character like a normal strategy game ruler. The Observer chooses where attention goes, which memories are seeded, which artifacts are disturbed, and which ripples enter the board.

Agents then interpret those inputs through their own remembered lives.

A small action may become a rumor.

A rumor may become a shared belief.

A belief may become a rule.

A rule may become history.

A history may become the foundation for another simulation inside the simulation.

---

## What Ripple Is Not

Ripple is not a direct adaptation of INTERVENTION.

Ripple is not a visual novel version of the chapters.

Ripple is not a claim that reality is fake.

Ripple is not a game where AI agents are described as literally conscious.

Ripple is not a replacement for the INTERVENTION repo.

Ripple is a playable system built from symbolic material.

The game should show the theory through play, not lecture the player about it.

---

## Core Play Shape

Ripple plays like a Civilization-style board viewed through an ARG archive.

The board is built from:

- rooms
- agents
- artifacts
- memories
- ripples
- laws
- factions
- nested simulations

The first playable direction is a 2D map with cards, panels, turn progression, and generated event logs.

The player should feel like they are moving across a living archive, not walking through a fixed level.

---

## Opening Modes

At the beginning of a new game, the player chooses how the board remembers its people.

### Mystery

The board chooses every agent's remembered life.

The player enters without knowing what shaped anyone.

### Vague

The player chooses A, B, or C memory seeds for each character without seeing the full meaning.

The player controls the beginning, but not the consequence.

### Experimental

The player sees the memory variants before choosing.

This mode is for testing conditions, replaying systems, and intentionally building different realities.

---

## The Living Board

Ripple should begin as a small board, not a giant open world.

The first demo target is:

**Ripple: The Boulder Build**

A small 2D board centered on the Boulder Room, a few connected rooms, a small cast of agents, active artifacts, memory seeds, and a turn loop.

The first demo only needs to prove one thing:

A small observed change can alter how agents remember, interpret, organize, and institutionalize reality.

---

## Source Material Boundary

For now, this repo still contains copied INTERVENTION material.

That material is temporary fuel for extraction, indexing, symbolic mapping, and agent construction.

Later, most direct INTERVENTION source content may be removed or relocated so Ripple can stand as its own game project.

Until then:

- do not overwrite source material
- do not convert chapters directly into levels
- do not hard-script story outcomes
- extract symbolism, room functions, artifacts, and character pressure
- preserve the original ARG material as source context only
