# ROOM SYMBOL MAP

_Status: First Pass | Source-To-Board Translation_

This map defines how copied source chapters can become symbolic rooms for Ripple.

The source stories are not levels.

The source stories provide room functions.

---

## Main Sequence Room Functions

### Prologue — Doorway Room

Function: entry, first contact, consent to observe.

### Chapter 01 — First Ripple Room

Function: origin architecture, adoption, first displacement, movement before understanding.

### Chapter 02 — Trigger Room

Function: gap between impulse and action.

Core risk: trigger mistaken for instruction.

### Chapter 03 — Hallway Room

Function: architecture making behavior sensible.

Core risk: output blamed while room remains invisible.

### Chapter 04 — Missed Layer Room

Function: visible complaint hiding grief, specificity, missed intervention.

Core risk: wrong visible layer becomes social reality.

### Chapter 05 — Shared Reality Room

Function: strangers forming a temporary room through delay, humor, and practical support.

Core risk: private failure story becomes public identity.

### Chapter 06 — False World Room

Function: useful language hardening into control.

Core risk: framework becomes cage.

### Chapter 07 — Observer Event Room

Function: a real shared event entering many rooms at once.

Core risk: massive reality erases room-level perception.

### Chapter 08 — Inheritance Room

Function: fatherhood, distance, small output, agency preserved.

Core risk: love becomes pressure.

### Chapter 09 — Kitchen Room

Function: rhythm under pressure, role clarity, nervous system stabilization.

Core risk: speed mistaken for progress.

### Chapter 10 — Signal Room

Function: creator releases output; receiver makes meaning.

Core risk: intention mistaken for arrival.

### Chapter 11 — Loop Room

Function: repeated pattern, morning architecture, small interruption.

Core risk: repeated failure becomes identity.

### Chapter 12 — Door Room

Function: access, not-your-door, visible opportunity hidden by observer reality.

Core risk: door becomes decorative.

### Chapter 13 — Echo Room

Function: phrase leaves sender, mutates through public systems.

Core risk: useful signal becomes merchandise, argument, or cage.

### Chapter 14 — Cosmic Room

Function: scale shift, cosmic room, observer humility.

Core risk: scale becomes dissociation or grandiosity.

### Chapter 15 — Boulder Room

Function: weight, consequence, obstruction made visible.

Core risk: named weight becomes law.

### Chapter 16 — Morning After Signal Room

Function: aftermath, grief, emotional residue, return from signal.

Core risk: signal becomes proof-demand instead of integration.

### Chapter 17 — Teodor Room

Function: return current, inheritance, cultural origin, accountability.

Core risk: origin becomes prison instead of context.

### Epilogue — Return Room

Function: loop closure, reader return, board hands control back.

Core risk: ending mistaken for solution.

## First Demo Room Set

Recommended for the first prototype:

1. Doorway Room
2. Trigger Room
3. Hallway Room
4. Boulder Room
5. Archive Room

The Boulder Room should be the center.

The other rooms should show how different symbolic functions react to one artifact ripple.

---

## Codex Translation Rule

When building data files, Codex should extract:

- room id
- source reference
- symbolic function
- core artifact
- pressure type
- active agents
- possible inputs
- possible outputs
- reality-layer risks

Codex should not copy whole chapters into gameplay text unless explicitly instructed.
