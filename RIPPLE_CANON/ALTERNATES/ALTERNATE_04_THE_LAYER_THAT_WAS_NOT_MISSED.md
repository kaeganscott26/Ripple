# ALTERNATE 04 — THE LAYER THAT WAS NOT MISSED

_Mirrors: Chapter 04 — The Missed Layer_

_Status: Ripple Canon | Alternate Path | Theory / Boundary / Unknown Current_

## Core Mirror

Chapter 04 names the missed layer: the visible conflict is not always the whole event.

This alternate asks what happens if the hidden layer is noticed in time.

The opposite ripple is early recognition.

## Alternate Story

The room sees the argument.

But this time it does not stop there.

It asks what the argument is covering.

A visible problem sits on the table, loud enough to attract every eye. The old room would treat that visible problem as the whole truth. The alternate room notices the shape of what is missing.

Mara hears the gap in testimony. She does not fill it with accusation. She marks it as unknown.

Jamal sees the form trying to close before the evidence is whole.

Maren feels the room getting tense and asks whether the tension belongs to the present moment or to something older.

Dev notices a door inside the door: the obvious path is not the only path.

Teodor / Scott realizes that a theory can miss the actual wound underneath it if it loves its own structure too much.

## Intervention Point

Someone says:

“We are not ready to decide. Something underneath this is still hidden.”

The room does not treat uncertainty as weakness.

It treats uncertainty as protection.

## Missed Intervention Point

The visible conflict becomes the entire story.

If missed, the hidden layer keeps moving under the floor and later returns as a stronger pressure.

## Ripple Event

A new board rule appears:

When a space feels too simple, check for a missed layer before resolving the turn.
