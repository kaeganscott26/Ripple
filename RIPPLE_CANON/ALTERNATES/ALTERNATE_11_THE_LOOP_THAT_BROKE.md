## 📖 Alternate Story

The room recognizes the pattern before the pattern completes.

The same pressure returns. The same sentence rises. The same path tries to choose the next turn because it has chosen so many turns before.

But this time, someone notices the repetition while there is still room to move.

[[Mara]] reads the loop as witness pressure. If the same moment keeps happening, the room needs to record what repeats.

[[Jamal]] reads the loop as form pressure. Repetition can become structure even when no one votes for it.

[[Maren]] reads the loop as mood pressure. A room can feel trapped because it knows the next scene before it happens.

[[Dev]] reads the loop as access pressure. A door can appear open while every path returns to the same space.

[[Teodor / Scott]] reads the loop as proof that naming a pattern does not fix it, but it creates the first possible breakpoint.

The board lets the character stop.

That stop is the event.

---

## 🟢 Intervention Point

Someone says:

> We have been here before, and that matters.

The room does not shame the repetition.

It marks the repetition as map data.

## 🔴 Missed Intervention Point

The room treats the repeated pattern as personality.

If missed, the loop becomes identity and the character forgets it was ever a path.
