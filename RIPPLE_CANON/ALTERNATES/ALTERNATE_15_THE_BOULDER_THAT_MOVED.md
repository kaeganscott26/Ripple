---

## 📖 Alternate Story

The Boulder appears in the room.

Everyone can see it.

The old path argues about what the Boulder means until the argument becomes another Boulder.

This time, someone asks a different question:

> Who is this weight changing?

[[Mara]] reads the question as witness correction. Seeing the Boulder is not enough if the room does not notice who has been moving around it.

[[Jamal]] reads the question as form correction. A named obstruction should not become permanent structure just because the room made a label for it.

[[Maren]] reads the question as safety. If the Boulder changes the room, the right answer is not only better theory. It is support.

[[Dev]] reads the question as access. A path blocked by the Boulder is not an equal path.

[[Teodor / Scott]] reads the question as responsibility. The tool is not finished if it only measures the weight and never helps the room respond.

The Boulder is not erased.

The room changes its relation to it.

## 🟢 Intervention Point
