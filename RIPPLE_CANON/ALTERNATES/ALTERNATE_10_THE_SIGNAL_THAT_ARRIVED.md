## 🧭 Core Mirror

[[Chapter 10]] follows signal: the thing sent, received, delayed, or changed by the path.

This alternate asks what happens when the signal arrives cleanly enough to be used.

The opposite ripple is **usable transmission**.

---

## 📖 Alternate Story

The signal leaves one room and reaches another before the room rewrites it.

There is still noise.

There is always noise.

But this time, the signal keeps enough shape to be recognized.

[[Mara]] reads the signal as witness: something was sent, and now there is evidence that it arrived.

[[Jamal]] reads the signal as form: once a message is received, the room can build a repeatable handle around it.

[[Maren]] reads the signal as mood: a clean arrival lowers pressure because the room no longer has to guess everything.

[[Dev]] reads the signal as access: the door between sender and receiver becomes functional for one turn.

[[Teodor / Scott]] reads the signal as architecture. A message is not just content. It is path, medium, timing, and receiver state.

The signal arrives, but arrival does not mean total understanding.

It only means the board can use it.

---

## 🟢 Intervention Point

Someone repeats the signal back accurately enough.

The room changes because the sender can stop treating the message as vanished.

---

## 🔴 Missed Intervention Point

The room reacts to the noise instead of the signal.

If missed, distortion becomes the event.

---

## 🌊 Ripple Event

The board creates a receipt mechanic:

> A signal does not require control of the receiver, but it can still be acknowledged.
