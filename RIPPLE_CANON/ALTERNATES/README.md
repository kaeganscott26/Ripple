## 🧭 Purpose

This folder contains the alternate canon layer for **Ripple**.

[[INTERVENTION ARG]] remains the canonical source archive. These alternates do **not** rewrite [[INTERVENTION]]. They create the playable mirror layer Ripple needs so the board can render branches, intervention points, missed intervention points, artifact redirects, and ripple events.

Each alternate mirrors one INTERVENTION chapter by reversing, delaying, redirecting, or exposing the chapter's central ripple.

The alternates are not chapters.

They are board-path canon.

---

## 🎲 Core Rule

A chapter shows what happened in the canonical room.

An alternate shows what could have happened if the opposite ripple took hold.

The game decides which version becomes real during a run.

```text
canonical chapter → alternate mirror → board space → dice event → artifact redirect → run log
```

## 🧠 Design Note

Ripple should not make the player decode a dashboard.

The board should teach the language by movement:

```text
roll → move → reveal → interpret → remember
```

A loop repeats.

A torus circulates.

A board remembers.
